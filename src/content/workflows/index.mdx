# Workflows

Workflows allow you to automate complex sequences of actions and tasks in the Makinari platform.

## Authentication

All API requests require an API key passed as a Bearer token in the `Authorization` header.

```bash
Authorization: Bearer YOUR_API_KEY
```

Explore the sidebar to see detailed documentation for each available workflow.
