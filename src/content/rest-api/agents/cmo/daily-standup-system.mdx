# API de Daily StandUp System Analysis de CMO

Esta API permite al agente CMO realizar un análisis comprensivo del estado del sistema, incluyendo configuración del sitio, estado de facturación, métricas de actividad y recomendaciones estratégicas para optimización y crecimiento.

## Endpoint

`POST /api/agents/cmo/dailyStandUp/system`

## Request Body

| Campo | Tipo | Requerido | Descripción |
|-------|------|-----------|-------------|
| site_id | string | Sí | Identificador único del sitio para analizar. Debe ser un UUID válido. |
| command_id | string | No | Identificador opcional del comando si forma parte de una secuencia existente. |

## Response

| Campo | Tipo | Descripción |
|-------|------|-------------|
| success | boolean | Indica si la solicitud fue exitosa. |
| data.command_id | string | Identificador único del comando creado. |
| data.summary | string | Resumen estratégico del análisis del sistema. |
| data.analysis_type | string | Tipo de análisis realizado ("system"). |
| data.system_data | object | Datos del sistema analizados. |
| data.system_data.site_status | string | Estado actual del sitio. |
| data.system_data.billing_status | string | Estado de facturación del sitio. |
| data.system_data.events_count | number | Número de eventos recientes (últimas 24 horas). |
| data.system_data.settings_count | number | Número de configuraciones del sitio. |

En caso de error:

| Campo | Tipo | Descripción |
|-------|------|-------------|
| success | boolean | false |
| error.code | string | Código de error (INVALID_REQUEST, AGENT_NOT_FOUND, DATA_ERROR, COMMAND_EXECUTION_FAILED, INTERNAL_SERVER_ERROR). |
| error.message | string | Descripción del error. |

## Características

- **Análisis Estratégico Completo**: Evalúa el estado de salud del sistema desde una perspectiva de negocio
- **Revisión de Facturación**: Analiza el estado de suscripciones y oportunidades de optimización financiera
- **Análisis de Configuración**: Identifica brechas críticas en la configuración del sistema
- **Métricas de Actividad**: Evalúa la actividad reciente del usuario y engagement
- **Recomendaciones Accionables**: Proporciona prioridades estratégicas y pasos siguientes claros
- **Evaluación de Riesgos**: Identifica riesgos potenciales y oportunidades perdidas

## Análisis Incluido

El agente CMO proporciona un análisis estructurado que incluye:

### 1. Evaluación de Salud del Sistema
- Calificación general del sistema (Verde/Amarillo/Rojo)
- Identificación de problemas críticos
- Evaluación de estabilidad operacional

### 2. Revisión Estratégica de Facturación
- Estado de suscripciones y facturación
- Identificación de preocupaciones financieras
- Oportunidades de optimización de ingresos

### 3. Análisis de Configuración y Setup
- Completitud de la configuración del sistema
- Brechas críticas que impiden el máximo rendimiento
- Mejoras de configuración con mayor impacto

### 4. Estado de Onboarding y Activación
- Porcentaje estimado de configuración completa
- Principales bloqueadores para activación completa
- Prioridades para mejorar la experiencia de onboarding

### 5. Recomendaciones de Acción Estratégica
- Top 3 prioridades estratégicas para la semana
- Quick wins implementables inmediatamente
- Cambios de configuración recomendados para impulsar crecimiento

### 6. Evaluación de Riesgos y Oportunidades
- Riesgos identificados en el estado actual del sistema
- Oportunidades perdidas por configuración incompleta
- Medidas preventivas recomendadas

## Casos de Uso

- **Daily Standups Ejecutivos**: Proporcionar resúmenes estratégicos para reuniones diarias
- **Análisis de Salud del Sistema**: Evaluación regular del estado operacional
- **Optimización de Configuración**: Identificar mejoras críticas de setup
- **Revisión de Billing**: Monitoreo estratégico del estado financiero
- **Planificación Estratégica**: Identificar prioridades y próximos pasos

## Validaciones

- **UUID Validation**: El `site_id` debe ser un UUID válido
- **Agent Validation**: Debe existir un agente CMO activo para el sitio
- **Data Validation**: Se valida la disponibilidad de datos del sistema

## English Documentation

### Daily StandUp System Analysis

Generates a comprehensive strategic analysis of system health for daily standup meetings, including site configuration, billing status, activity metrics, and strategic recommendations.

### Endpoint

```http
POST /api/agents/cmo/dailyStandUp/system
```

### Request Body

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|

### Strategic Analysis Framework

The CMO agent provides analysis across six key strategic areas:

1. **System Health Assessment** - Overall system status and critical issues
2. **Strategic Billing Review** - Financial health and optimization opportunities  
3. **Configuration Analysis** - Setup completeness and critical gaps
4. **Onboarding Status** - Activation blockers and completion percentage
5. **Strategic Recommendations** - Top priorities and quick wins
6. **Risk Assessment** - Current risks and missed opportunities

This comprehensive analysis serves as the foundation for executive briefings and strategic decision-making.
