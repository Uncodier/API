# Uncodie API

## La API de análisis web impulsada por IA

Uncodie es una API REST que te permite analizar sitios web utilizando modelos avanzados de inteligencia artificial. Obtén insights detallados sobre estructura, contenido y experiencia de usuario de cualquier sitio web a través de endpoints simples y bien documentados.

## Características de la API

- **Análisis Semántico** - Obtén análisis profundo del contenido y estructura de sitios web
- **Múltiples Modelos de IA** - Flexibilidad para elegir entre Claude 3.5, GPT-4 o Gemini 1.5
- **Rate Limits Generosos** - Hasta 100 solicitudes por minuto en planes básicos
- **Respuestas Estructuradas** - Datos en formato JSON limpio y consistente
- **Soporte WebSocket** - Para análisis en tiempo real y streaming de respuestas

## Inicio Rápido

### Autenticación

```javascript
// Obtén tu API key en https://uncodie.com/dashboard
const UNCODIE_API_KEY = 'tu-api-key';

// Incluye la key en los headers de tus requests
const headers = {
  'Authorization': `Bearer ${UNCODIE_API_KEY}`,
  'Content-Type': 'application/json'
};
```

### Análisis Básico

```javascript
// Ejemplo de análisis básico
async function analyzeWebsite(url) {
  const response = await fetch('https://api.uncodie.com/v1/analyze', {
    method: 'POST',
    headers,
    body: JSON.stringify({ 
      url,
      model: 'claude-3' // Opcional, por defecto usa Claude 3
    })
  });
  
  const analysis = await response.json();
  return analysis;
}
```

### Análisis con Streaming

```javascript
// Ejemplo de análisis con respuestas en tiempo real
const ws = new WebSocket('wss://api.uncodie.com/v1/analyze/stream');
ws.onmessage = (event) => {
  const update = JSON.parse(event.data);
  console.log('Nuevo insight:', update);
};

ws.send(JSON.stringify({
  url: 'https://ejemplo.com',
  apiKey: UNCODIE_API_KEY
}));
```
