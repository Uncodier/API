# API de Site Analyzer

Esta sección proporciona documentación detallada sobre todos los endpoints de la API de Site Analyzer.

## Endpoints disponibles

- [/api/analyze](/api/analyze) - Analiza un sitio web y devuelve información detallada
- [/api/ai](/api/ai) - Interactúa con modelos de IA para análisis avanzados
- [/api/site](/api/site) - Gestiona información de sitios web
- [/api/conversation](/api/conversation) - Gestiona conversaciones con la IA
- [/api/hello](/api/hello) - Endpoint de prueba

## Autenticación

Actualmente, la API no requiere autenticación para uso básico, pero tiene límites de uso para prevenir abusos.

## Formatos de respuesta

Todas las respuestas de la API están en formato JSON.
